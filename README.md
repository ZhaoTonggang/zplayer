# zplayer

#### 介绍
免费，开源，可商用的h5网页播放器

原生JS编写，不依赖任何第三方组件

兼容性强，性能稳定，体积小，可跨域

#### 版权
开源，免费，可商用

#### 使用说明
1. 在页面内引入 css/zplayer.min.css和js/zplayer.min.js
